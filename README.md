# Pong-Game
My first project in C.

Controls 
Left racket: W (up) and S (down). 
Right racket: Up arrow and Down arrow. 

How to build the game 
1. Make sure the Raylib library is installed. 
2. Make sure the Raylib library is installed. 

`gcc pong.c -o pong -lraylib -lm` 

3. Start the game: './pong'
